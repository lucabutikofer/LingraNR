# LingraNR
R Interface to [LINGRA-N Tool](https://widgets.figshare.com/articles/11359613/embed?show_title=1), grassland productivity model.

***
### Installation

Install with:
```R
devtools::install_github('lucabutikofer/LingraNR')
```
